# API_REST_Gestion_de_Tareas_Pro_Avanzado
API RESTful desarrollada con Spring Boot para la gestión de tareas (To-Do List) con enfoque profesional. Implementa arquitectura en capas, DTOs, validaciones con Bean Validation, manejo global de errores, paginación y documentación con Swagger/OpenAPI. Proyecto orientado a portfolio backend y aplicaciones frontend-ready.
